My project on Airline ticket management system is all about booking tickets,
viewing updates on reservation of the ticket booked in an online manner. The
project simplifies the operation to a significant extent. Although the
operations done are identical to railway or bus reservation, it familiarizes the
user with the backend working of the system that is facilitating him/her.
Almost all the operations are included in this project barring ticket
cancellation.

HARDWARE AND SOFTWARE REQUIREMENT
1) HARDWARE
    • PC
2) SOFTWARE
    • PYTHON (latest version of IDLE or any other development suite)
    • MYSql Python Connector
    • MySQL(v 8.0 or above )
